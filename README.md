## Getting started

### Dependencies
If you want to edit the files in this repo, you must install the **UC Irvine Machine Learning Repository** Python library. This will load the dataset used in this activity, without having to worry about filepaths or other configuration. **The dataset file is included in this repo for convenience, but the Notebook file accesses the dataset through the UCI library, NOT the repo file**

> You can modify the program to accept a filepath if you want to manually install the dataset, but this will take more involved editing of the file

**Pandas**, **matplotlib**, **Seaborn** and **scikit-learn** are additonal packages needed for this program, but you may have some or all of these already installed, as they are very common.

To easily install everything you need to run the code in the notebook, you can simply change `InstallOption` in the first code block to `True`, then run that block. **You only need to do this once**. *(the variable is there to prevent accidential redownloading of packages when using the "Run All" option)*. Alternatively, you can manually install these packages on your device.

> *Installing the packages will not work on most online code editors. However, the code output should already be displayed below each block*

### Jupyter Notebook

All the code behind this assignment is stored on one Jupyter Notebook (.ipynb) file. Most IDEs/code editors that support Python files support this format, it can be uploaded to [Jupyter Notebook's web-based IDE](https://jupyter.org/try-jupyter/lab/), or even viewed on this repo.

Code Cells can may be pre-ran with their results shown below, but you can run them again using the "run" or "run all" buttons in your IDE. 

Some code blocks modify the dataset, some visualize or output data, and some do both! Each block is one step in the process outlined by the project discription, and is clearly explained.
